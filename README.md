# 📦 Final Project — React.js

Proyek akhir yang dibangun menggunakan **React.js** dengan fitur CRUD dan integrasi API.

## ✨ Fitur

- **React.js** — Frontend library
- **CRUD Operations** — Create, Read, Update, Delete
- **API Integration** — Terhubung dengan backend/server
- **Tailwind CSS** — Styling modern

## 🛠️ Tech Stack

- **React.js** — Frontend library
- **JavaScript** — Bahasa utama
- **Tailwind CSS** — Styling
- **Axios** — HTTP client
- **Netlify** — Deployment

## 🚀 Cara Menjalankan

```bash
npm install
npm start
```
